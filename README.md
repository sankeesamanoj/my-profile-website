## 🚀 Getting Started

### 1. Clone or Download

```bash
git clone https://github.com/sankeesamanoj/portfolio.git
```

Or simply download the ZIP and extract it.

### 2. Add Your Photo

Place your profile photo in the same folder and name it exactly:

```
manoj.jpg
```

### 3. Open in Browser

No server or installation needed. Just open the file directly:

```
Double-click index.html
```

Or right-click → **Open with** → your browser.
